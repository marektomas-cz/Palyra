Inspect and configure model providers and defaults

Usage: palyra models [OPTIONS] <COMMAND>

Commands:
  status           Show the effective model-provider configuration
  list             List configured and available model catalog entries
  test-connection  Verify model-provider credentials and connectivity
  discover         Discover live provider models with registry fallback
  explain          Explain model routing and provider candidate selection
  set              Set the default chat model
  set-embeddings   Set the default embeddings model

Options:
  -h, --help  Print help

Global Options:
      --profile <PROFILE>

      --expect-profile <EXPECT_PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)
      --log-level <LOG_LEVEL>
          [default: info] [possible values: error, warn, info, debug, trace]
      --output-format <OUTPUT_FORMAT>
          [default: text] [possible values: text, json, ndjson]
      --plain

      --no-color

      --allow-profile-mismatch

      --allow-strict-profile-actions
