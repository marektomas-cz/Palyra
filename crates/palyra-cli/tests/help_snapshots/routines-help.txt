Manage unified automation routines across schedules and event triggers

Usage: palyra routines [OPTIONS] <COMMAND>

Commands:
  status
  list
  show
  upsert                [aliases: apply]
  create-from-template
  enable
  disable
  run-now
  test-run              [aliases: replay]
  logs                  [aliases: runs]
  dispatch
  delete                [aliases: rm]
  templates
  schedule-preview
  export
  import

Options:
  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>
          Use the named CLI profile from the local profile registry

      --expect-profile <EXPECT_PROFILE>
          Require the resolved active profile name to match this value

      --config <CONFIG_PATH>
          Use this palyra.toml config file

      --state-root <STATE_ROOT>
          Use this runtime state root directory

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          Set diagnostic log verbosity

          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          Select text, JSON, or NDJSON output for automation

          [default: text]
          [possible values: text, json, ndjson]

      --plain
          Disable styled terminal output

      --no-color
          Disable ANSI color output

      --allow-profile-mismatch
          Allow commands to continue when the selected profile differs from context

      --allow-strict-profile-actions
          Permit strict profile-scoped actions after explicit profile checks

Examples:
  palyra routines list
  palyra routines create-from-template --template-id heartbeat
  palyra routines upsert --name "Daily report" --prompt "Summarize incidents" --trigger-kind schedule --natural-language-schedule "every weekday at 9"
  palyra routines schedule-preview "každý pracovní den v 9"

Discoverability:
  `routines` is the first-class automation surface. Use `cron` only for schedule-only compatibility flows.
