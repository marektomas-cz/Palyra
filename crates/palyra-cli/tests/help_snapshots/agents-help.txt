Manage daemon agent registry, bindings, and defaults

Usage: palyra agents [OPTIONS] <COMMAND>

Commands:
  list         List registered agents
  show         Show one registered agent
  bindings     List session, principal, and channel agent bindings
  bind         Bind an agent to a session or operator context
  unbind       Remove an agent binding from a session or operator context
  set-default  Set the default agent
  create       Create a registered agent
  delete       Delete a registered agent
  identity     Resolve the effective agent for an operator context

Options:
  -h, --help  Print help

Global Options:
      --profile <PROFILE>

      --expect-profile <EXPECT_PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)
      --log-level <LOG_LEVEL>
          [default: info] [possible values: error, warn, info, debug, trace]
      --output-format <OUTPUT_FORMAT>
          [default: text] [possible values: text, json, ndjson]
      --plain

      --no-color

      --allow-profile-mismatch

      --allow-strict-profile-actions
