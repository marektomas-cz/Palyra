# Client Applications

This directory contains client-facing applications referenced by the architecture.

## Directories

- `apps/ios`: iOS node app scaffold and Swift lint baseline.
- `apps/android`: Android node app scaffold and Kotlin lint baseline.
- `apps/desktop`: desktop control center v1 (Tauri sidecar supervisor + health cockpit).
- `apps/browser-extension`: local MV3 extension artifact for M37 relay workflows.
- `apps/web`: web console scaffold and web lint baseline.
